# Git Log Explorer

一个功能强大的VSCode Git日志管理插件，提供直观的Git历史查看、分支管理和提交操作功能。

## ✨ 主要功能

### 🌿 智能分支管理
- **侧边栏Git视图**：专用的Git Log Explorer侧边栏，一键访问
- **分支/标签切换**：支持本地分支、远程分支和标签的快速切换
- **智能搜索**：可搜索的分支/标签选择器，支持关键字筛选
- **远程数据同步**：一键刷新远程分支和标签信息

### 📊 提交历史查看
- **完整提交信息**：显示完整的40位commit ID、作者、日期和提交消息
- **优雅的界面**：清晰的提交列表，支持展开查看详细信息
- **文件差异查看**：点击文件可查看前后对比差异
- **智能筛选**：支持按作者和提交消息进行筛选

### ⚖️ 分支比较功能
- **可视化比较**：专业的分支/标签比较界面
- **智能筛选选项**：
  - 隐藏相同提交：只显示各分支独有的提交
  - 作者筛选：按指定作者筛选比较结果
- **左右对比显示**：清晰的并排比较视图
- **记忆功能**：自动记住上次比较的分支选择

### 🔀 交互式操作
- **多选提交**：支持选择多个提交进行批量操作
- **交互式变基**：模拟rebase操作（合并、编辑、重排序、删除）
- **文件级操作**：双击文件查看详细差异

## 🚀 快速开始

### 安装
1. 在VSCode扩展市场搜索 "Git Log Explorer"
2. 点击安装
3. 重启VSCode

### 使用
1. 打开包含Git仓库的项目
2. 在侧边栏找到Git Log Explorer图标（🌿）
3. 点击打开插件界面
4. 开始探索你的Git历史！

## 📖 详细使用指南

### 基本操作

#### 查看提交历史
- 插件会自动检测当前Git仓库并显示当前分支的提交历史
- 每个提交显示：完整commit ID、作者、日期、提交消息
- 点击提交可展开查看修改的文件列表

#### 切换分支/标签
- 使用顶部的分支选择器
- 支持输入搜索：直接输入分支名称并按回车
- 下拉选择：点击下拉箭头选择现有分支或标签
- 支持本地分支、远程分支和标签

#### 筛选提交
- **作者筛选**：在"👤"输入框中输入作者名称
- **消息筛选**：在"💬"输入框中输入关键字
- **清除筛选**：点击"清除"按钮重置所有筛选条件

### 高级功能

#### 分支比较
1. 点击"比较"按钮打开比较模态框
2. 选择起始分支/标签和结束分支/标签
3. 可选设置：
   - ☑️ 只显示不同的提交：隐藏commit message相同的提交
   - 作者筛选：只显示指定作者的提交
4. 点击"开始比较"查看结果

#### 文件差异查看
- 在提交详情中双击任意文件
- 自动打开VSCode的差异查看器
- 显示文件在该提交前后的变化

#### 交互式变基（模拟）
1. 使用复选框选择多个提交
2. 点击"🔀 合并选中"按钮
3. 选择操作类型：合并、编辑、重排序、删除

#### 远程数据刷新
- 点击"🔄"刷新按钮获取最新的远程分支和标签
- 自动处理权限问题，无权限时使用本地缓存

## ⚙️ 技术特性

### 性能优化
- **批量Git命令**：使用`git for-each-ref`批量获取分支和标签信息
- **智能缓存**：避免重复的Git操作
- **分页渲染**：大量分支时限制显示数量，提升界面响应速度
- **异步加载**：UI和数据加载分离，提升用户体验

### 智能引用解析
- **优先远程版本**：自动优先使用远程分支的最新版本
- **回退机制**：远程不可用时自动回退到本地版本
- **标签支持**：完整支持Git标签的查看和比较

### 错误处理
- **网络容错**：网络问题时优雅降级
- **权限处理**：SSH权限问题时提供友好提示
- **数据验证**：完整的输入验证和错误提示

## 🛠️ 开发信息

### 技术栈
- **前端**：TypeScript + HTML/CSS + JavaScript
- **后端**：VSCode Extension API + simple-git
- **架构**：WebviewProvider + Git Service

### 系统要求
- VSCode 1.74.0 或更高版本
- Git 2.0 或更高版本
- Node.js 16.x 或更高版本
