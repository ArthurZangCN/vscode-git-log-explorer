# 安装和测试指南

## 🚀 快速安装

### 安装插件
1. 下载最新的 `cursor-git-log-explorer-1.0.0.vsix` 文件
2. 在VSCode中按 `Ctrl+Shift+P` 打开命令面板
3. 输入 `Extensions: Install from VSIX...`
4. 选择下载的 `.vsix` 文件
5. 重启VSCode

### 打开插件
1. 点击左侧活动栏的Git图标 🌿
2. 查看 "Git Log Explorer" 面板

## 🐛 已知问题解决

### 问题1: 分支为空
如果分支列表为空，检查：
- 是否在Git仓库中
- 当前目录是否有.git文件夹
- 尝试刷新插件（重新打开面板）

### 问题2: 比较功能报错
如果比较时出现错误：
- 确保选择的分支/标签存在
- 检查分支名称是否正确
- 尝试使用本地分支而不是远程分支

### 问题3: 搜索不生效
如果搜索功能不工作：
- 确保输入了有效的关键字
- 检查是否有匹配的分支/标签
- 尝试清空输入框重新搜索
